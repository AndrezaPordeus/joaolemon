# 🍋 Tema João Lemon - Curso WordPress do Zero Essencial

Este repositório contém o código e os arquivos do tema desenvolvido durante as aulas do curso **[WordPress do Zero Essencial](https://www.udemy.com/)**, ministrado pelo professor **Bruno de Sousa** na plataforma Udemy.

## 📖 Sobre o Projeto

O objetivo deste repositório é documentar o aprendizado e armazenar o progresso prático do curso. Aqui você encontrará a estrutura de um tema WordPress criado do zero (Tema João Lemon), compreendendo a hierarquia de arquivos do WordPress, a integração de scripts e estilos, e a construção de páginas personalizadas.

## 🚀 Tecnologias Utilizadas

As seguintes tecnologias e ferramentas foram abordadas e utilizadas neste projeto:

- **WordPress** (CMS)
- **PHP** (Lógica do tema e integração com o núcleo do WP)
- **HTML5 & CSS3** (Estruturação e estilização das páginas)
- **JavaScript** (Interatividade e animações da interface)
- **Anime.js** (Biblioteca de animação JavaScript utilizada no projeto)

## ⚙️ Como Instalar e Rodar

Para visualizar este tema em funcionamento na sua máquina local, siga os passos abaixo:

1. Certifique-se de ter um ambiente local configurado (como XAMPP, LocalWP ou WampServer) com o WordPress instalado.
2. Faça o clone deste repositório para dentro da pasta de temas do seu WordPress:
   ```bash
   cd wp-content/themes/
   git clone <URL_DO_SEU_REPOSITORIO> joaolemon
   ```
3. Acesse o painel administrativo do seu WordPress (`http://localhost/seu-site/wp-admin`).
4. Vá até a aba **Aparência > Temas**.
5. Procure pelo tema e clique em **Ativar**.
